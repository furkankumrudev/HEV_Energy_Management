# Security Policy

## Supported versions

Only the latest version on the `main` branch is currently supported.

## Reporting a vulnerability

Please do not disclose security vulnerabilities in a public issue.

Use GitHub's private vulnerability reporting feature to send the details
privately to the repository maintainer. Include reproduction steps, affected
files, and the expected impact when possible.

This project is an academic supervisory-controller prototype. It is not
intended for direct deployment in safety-critical vehicle systems.
